# VCD Parser & Waveform Analyzer

A minimal Python tool that parses Value Change Dump (VCD) files generated by Icarus Verilog and extracts signal activity statistics — without relying on GTKWave or any VCD library.

## What It Does

- Parses the VCD header to map internal symbols to real signal names
- Extracts a timestamped list of every signal value change
- Computes toggle counts per signal (a basic switching activity metric used in power analysis)

## Project Structure

```
vcd_parser/
├── vcd_parser.py   # Core parser (reusable, no side effects)
├── analyze.py      # Run this — computes stats and prints results
└── wave.vcd        # Your Icarus Verilog simulation output
```

## Requirements

- Python 3.x
- No external libraries needed for the parser core

## Usage

1. Run your Icarus Verilog testbench with `$dumpfile` and `$dumpvars` enabled
2. Copy the output `wave.vcd` into this folder
3. Run:

```bash
python analyze.py
```
